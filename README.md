# Xiaomi Pad 5 Pro ZXL 官改包修復
- 開機自動啟用鍵盤
- SoC 名稱更正為 `SM8250-AC`

## 備註
- 鍵盤是默認啟用，所以不想用鍵盤的話就打開側滑欄呼出On Screen鍵盤
- 僅為此包做適配 （其他包刷了也沒意義）
- 硬體檢測軟件如果還是寫SM8250，清數App據再開啟即可
